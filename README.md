## ðŸš€ Getting Started

To run the demo:

1. Clone this repository.
2. Open `RemoteImageDemo.xcodeproj` in Xcode.
3. Make sure the [RemoteImage](https://github.com/AbrorbekSh/RemoteImage) dependency is resolved.
4. Run the app on a simulator or device.
